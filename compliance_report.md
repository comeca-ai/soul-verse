# Compliance Report

## Propósito
O app "SoulVerse" tem um propósito claro e benéfico: fornecer conforto espiritual rápido. Não é deceptivo nem viola políticas de conteúdo.

## Qualidade
- O app usa uma fonte de dados confiável (não alucina versículos).
- O design é premium e responsivo.
- Não existem "hello world" placeholders visíveis.

## Privacidade
- Mínimo de dados coletados (apenas query de sentimento efêmera).
- Nenhuma PII solicitada.
- CSP restritivo configurado.

## Tooling
- 1 Tool apenas (`get_bible_verse`), focada e bem descrita.
- Nomes e descrições cristalinas para o modelo.
