# Submission Assets Checklist

Os seguintes itens devem ser capturados manualmente para a submissão no console:

1. **App Icon**: 512x512px (Sugerido: Uma cruz minimalista ou um livro aberto com luz suave).
2. **Screenshots**:
   - [ ] **Fluxo Principal**: Usuário diz "Estou ansioso" -> Bot responde com texto de apoio -> Widget exibe o card de Filipenses 4:6.
   - [ ] **Card Visual**: Screenshot focado apenas no Card renderizado para mostrar a qualidade do design ("Rich Aesthetics").
3. **Privacy Policy Link**: O rascunho está em `security_privacy.md`, mas para submissão real, hospedar em uma URL pública (ex: GitHub Gist ou Vercel).
