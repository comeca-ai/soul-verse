# SoulVerse Final Submission Packet

## App name

`SoulVerse`

## Subtitle

`Find comforting Bible verses`

## Short description

`Busca versiculos biblicos de conforto com base no sentimento do usuario ou em uma referencia explicita.`

## Full description

`SoulVerse ajuda o usuario a encontrar versiculos biblicos de conforto para momentos de ansiedade, tristeza, medo, cansaco, culpa, solidao e incerteza. O app identifica um tema emocional principal, seleciona uma referencia biblica apropriada, consulta o texto exato em uma API biblica publica e exibe o resultado em um card visual. O app tambem pode buscar o texto de uma referencia biblica explicita informada pelo usuario.`

## Tool behavior summary

- both tools are read-only
- both tools return devotional content only
- the app does not send the user's emotional text to the Bible API, only the chosen or explicit Bible reference
