# Privacy Policy for SoulVerse

Effective date: 2026-03-19

## 1. Overview

SoulVerse is a ChatGPT app that provides biblical verses for comfort based on the user's emotional context or an explicit Bible reference. It retrieves verse text from a public Bible API and presents the result in a visual card.

## 2. Data we process

SoulVerse may process:

- a short description of the user's emotional state
- an explicit Bible reference provided by the user
- minimal technical metadata required to operate the service securely

SoulVerse does not request passwords, financial information, health records, identity documents, or unrelated personal information.

## 3. Why we use this data

We use the submitted information only to:

- determine an appropriate Bible reference for spiritual comfort
- retrieve the exact verse text from a public Bible API
- render the verse in the SoulVerse widget
- maintain service security and reliability

## 4. Third-party services

SoulVerse currently uses a public Bible API to retrieve verse text. The request sent to the API contains the Bible reference only. We do not intentionally send the user's full emotional description to that external API.

## 5. Data retention

SoulVerse is designed for minimal retention. Emotional descriptions and requested references are used only for the active request flow, plus limited technical debugging, abuse prevention, and legal compliance if necessary.

## 6. Data sharing

We do not sell personal data. We may share limited information only with infrastructure or service providers required to run the app, or when legally required.

## 7. Security

We use reasonable technical and organizational safeguards to reduce the risk of unauthorized access, misuse, or disclosure.

## 8. Children

SoulVerse is not designed for or marketed to children under 13.

## 9. Changes

We may update this policy from time to time. The latest version should remain available at the published privacy policy URL.
