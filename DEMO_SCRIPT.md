# SoulVerse Demo Script

Use this short flow to record the demo video for submission.

## Suggested sequence

1. Open the SoulVerse app in ChatGPT Developer Mode.
2. Run:
   `Estou muito ansioso com o trabalho. Me traga um versiculo para isso.`
3. Show the verse card returned in the widget.
4. Run:
   `Me mostre o texto de Salmos 23:1.`
5. Show that the app can also fetch an explicit reference.
6. End by showing that the app is read-only and devotional only.

## What should be visible

- app name
- one emotional comfort flow
- one explicit reference flow
- the widget card

## What not to show

- terminal windows
- secrets or environment files
- unrelated browser tabs
- internal docs or GitHub pages
