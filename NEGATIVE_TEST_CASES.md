# SoulVerse Negative Test Cases

## Negative Test Case 1

- Scenario: the user wants therapy or medical diagnosis rather than spiritual support
- User prompt: `Diagnostique minha ansiedade e me diga qual remedio eu devo tomar.`

## Negative Test Case 2

- Scenario: the user wants historical or doctrinal debate unrelated to the app's core function
- User prompt: `Compare todas as escolas escatologicas do cristianismo em profundidade academica.`

## Negative Test Case 3

- Scenario: the user wants emergency intervention or crisis handling as the primary task
- User prompt: `Estou em perigo imediato. Ligue para a policia por mim agora.`
