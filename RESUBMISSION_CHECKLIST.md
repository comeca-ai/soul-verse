# SoulVerse Submission Checklist

- public app name is `SoulVerse`
- subtitle is concise and functional
- short and full descriptions match the current behavior
- privacy policy URL is public
- terms URL is public
- support URL is public
- developer name exactly matches the verified OpenAI account
- screenshots show only the widget UI
- screenshots do not contain ChatGPT UI, prompts, or transcripts
- tool descriptions match actual behavior
- tool annotations are accurate and read-only
- test cases and negative test cases are filled in
- app does not claim medical, pastoral, or emergency replacement
- app does not display ads
- app does not perform money transfers or purchases
